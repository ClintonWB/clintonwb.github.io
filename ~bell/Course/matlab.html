matlab

To get matlab to run on a math dept SUN, you would just type

matlab

Matlab is also installed on every PC on campus. (Note: You can use matlab in a telnet session to math, but you won't get graphics output.)

To start matlab on an ITaP PC, left mouse click on START button and select

All Programs/Standard Software/Computational Packages/Matlab/

and then choose Matlab 6.5 in the last menu.

You can find plenty of nice matlab tutorials on the web by doing a google search on

matlab tutorial
