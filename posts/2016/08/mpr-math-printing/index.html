Printing over SSH

Date Mon 22 August 2016 By Clinton Bradford Category howto. Tags sh purduemathdept

A common frustration here in the Purdue math department is having to use a computer wired into the network to send print jobs. We would like to

Here is my solution: Sending the print via the server Banach.

cat [path to file] | ssh [username]@banach.math.purdue.edu "lpr -P [printer]"

For example, if I wanted to print a file paper.pdf in the Documents directory, I would open a command prompt and type

cat Documents/paper.pdf | ssh bradford@banach.math.purdue.edu "lpr -P math737bw"

To simplify the process, I save the following as the file mpr.sh in my path

#!/bin/bash
if [ $# -ne 1 ]; then
    echo $0: usage: $0 name
    exit 1
fi
cat $1 | ssh bradford@banach.math.purdue.edu "lpr -P math737bw"

Allowing me to simply invoke

mpr file.pdf

If you wish to use that file, change the username to your math department username. Change the printer to a printer of your choice.

To list available printers, run on Banach:

:::bash
lpstat -p -d
